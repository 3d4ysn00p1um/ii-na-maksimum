Анатомия идеального промпта

Хороший промпт — не случайность. Это структура из четырёх элементов. Освой их один раз — и ты будешь писать промпты которые работают с первого раза, каждый раз.

Из чего состоит промпт

Промпт-инженеры не придумывают каждый запрос с нуля. Они используют проверенную формулу из четырёх компонентов. Разберём каждый:

🎭 Роль
Компонент 1
Кем должен быть ИИ?
Роль задаёт экспертизу, тон и угол зрения. ИИ буквально "надевает маску" специалиста и отвечает из его позиции. Без роли — ответ будет общим и поверхностным.
→ "Ты — опытный маркетолог с 10 годами опыта в B2B"
→ "Ты — строгий редактор делового издания"
→ "Ты — дружелюбный HR-специалист стартапа"
🎯 Задача
Компонент 2
Что нужно сделать?
Конкретное действие — написать, проанализировать, переработать, придумать. Чем точнее задача — тем точнее результат. Избегай размытых глаголов типа "помоги" или "расскажи".
→ "Напиши 5 вариантов заголовка для лендинга"
→ "Проанализируй этот текст на ошибки логики"
→ "Сократи этот абзац вдвое, сохранив смысл"
📋 Контекст
Компонент 3
Что нужно знать?
Всё что ИИ должен учесть: аудитория, продукт, ограничения, предыстория. Это самый недооценённый компонент — именно он делает ответ релевантным твоей ситуации.
→ "Аудитория: предприниматели 30–45 лет. Продукт: CRM-система"
→ "Текст уже существует, нужно его улучшить, а не переписывать"
→ "Тон должен быть как в Forbes, не как в Instagram"
📐 Формат
Компонент 4
Как должен выглядеть результат?
Длина, структура, стиль подачи. Без этого ИИ сам решает как оформить ответ — и часто угадывает неправильно. Укажи что именно ты хочешь получить.
→ "Ответ в виде маркированного списка, не более 7 пунктов"
→ "3–4 абзаца, без подзаголовков, деловой стиль"
→ "Только финальный текст — без объяснений и комментариев"
💡
Не все четыре нужны всегда. Для простых задач достаточно задачи + формата. Для сложных — все четыре. Главное — понимать зачем каждый компонент нужен.

Плохой промпт vs Хороший промпт

Посмотри как одна и та же задача выглядит без структуры и со структурой:

❌ Без структуры
Напиши мне описание продукта
✓ Со структурой
Роль Ты — копирайтер с опытом в e-commerce.

Задача Напиши описание продукта для карточки товара.

Контекст Продукт: беспроводные наушники премиум-класса. ЦА: музыканты и аудиофилы 25–40 лет. USP: 40 часов работы, ANC, LDAC.

Формат 3–4 предложения. Акцент на эмоцию и качество звука. Без технического жаргона.

Типичные ошибки — и как их исправить

🌫️

Слишком расплывчатая задача

"Помоги мне с текстом" — ИИ не знает что именно делать.

→ Замени на конкретное действие: "Сократи этот текст вдвое, сохранив ключевые аргументы"
🫥

Нет контекста об аудитории

ИИ не знает для кого пишет — и пишет для всех сразу, то есть ни для кого.

→ Всегда указывай: кто читатель, какой у него уровень, что он уже знает
📏

Не указан формат

ИИ напишет большой полотенчатый текст там, где нужен список из 5 пунктов.

→ Добавь: "Ответ в виде нумерованного списка, максимум 5 пунктов"
🎭

Нет роли для сложных задач

Без роли ИИ отвечает как "средний специалист" — без экспертной глубины.

→ Дай конкретную роль: не просто "эксперт", а "CFO компании с оборотом $10M"
